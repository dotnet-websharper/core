This is a minor release of WebSharper 10.1. NuGet version `10.1.5.674`.

Templates for .NET Core SDK: `dotnet new install WebSharper.Templates@10.1.5.674`

## Enhancements
* #1222 Added proxies for `IAsyncDisposable`, `IAsyncEnumerable`, and `ValueTask`, and C# compiler support for translating `await using`, `await foreach`, and `async` generator methods returning an `IAsyncEnumerable`.
* #1530 Added proxies for `use!` and `and!` inside F# `task` computation expressions.

## Fixes
* #1582 Fix infinite loop when copying files to output directory with `extra.files` if a pattern happens to match a file within the output folder. Now files inside output folder are always excluded.
