This is a minor release of WebSharper 10.1. NuGet version `10.1.3.672`.

Templates for .NET Core SDK: `dotnet new install WebSharper.Templates@10.1.3.672`

## Enhancements
* #1583 Importing non-JavaScript resources is now possible. Example:
```fsharp
[<Import("default", "./button.module.css"); Inline "$import.button">]
let buttonCss = X<string> 
```
To use `button.module.css` with `esbuild`, you will also need a plugin. Run `npm install --save-dev esbuild-css-modules-plugin` then edit `esbuild.config.mjs` to have:
```javascript
import cssModulesPlugin from "esbuild-css-modules-plugin";
//...
    var options =
    {
      //...
      plugins: [cssModulesPlugin()]
    };
```

## Fixes
* #1584 SPA (or BundleOnly) projects now also unpack any non-JavaScript embedded resources.
