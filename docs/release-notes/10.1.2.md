This is a minor release of WebSharper 10.1. NuGet version `10.1.2.671`.

Templates for .NET Core SDK: `dotnet new install WebSharper.Templates@10.1.2.671`

* Updated `System.Text.Json` dependency to version `10.0.0` for better performance and security.
