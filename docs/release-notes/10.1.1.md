This is a minor release of WebSharper 10.1. NuGet version `10.1.1.669`.

Templates for .NET Core SDK: `dotnet new install WebSharper.Templates@10.1.1.669`

## Enhancements
* #1579 Added missing `Map.count` proxy.
* #1580 A JavaScript output optimization: `let` declarations for mutable variables are now on a single line instead of separate lines.

## Fixes
* #1525 Fix `MISSINGVAR` error caused by erroneously dropping variaable declarations.
* #1578 Fix invalid inlining optimization involving mutable variables.
