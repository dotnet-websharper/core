This is a major release of WebSharper. NuGet version `10.1.0.666`.

Templates for .NET Core SDK: `dotnet new install WebSharper.Templates@10.1.0.666`

## Enhancements
* #1574 Add `debugger` keyword handling. Use `JS.Debugger` to insert it.
* #1568 Turn on ReadyToRun for compilers for Windows and glibc-based Linux distributions
* #1573 Update `WebSharper.AspNetCore` to `net5.0`. No more dependence on deprecated `IHostingEnvironment` interface.
* #1566 Better time measurement and download resources logging. New MSBuild property `WebSharperLogImportance`
* #1559 Support `.mjs` and `.cjs` resources
* #1565 Translate server-side expressions in F# top-level actions (module-bound `do`) or `fsi` submissions

## Fixes
* #1558 Fix SPA resource unpacking and `extra.files` handling
* #1563 Bump basic-ftp from 5.0.5 to 5.2.0
